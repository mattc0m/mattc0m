Helpful Hints

Routines

A Routine is a scheduled backup event that can be setup hourly, daily, weekly, or monthly. Each Routine has its own schedule, storage provider, and set of rules.

Storage Provider

You can add multiple storage providers, even from the same service. When you create a Backup Routine, it must save to a single Storage Provider.
